Cool Websites on the internet
The (not yet) ultimate list

This is a list of cool websites that I have found over my time in the internet. You can find all sorts of things here, from tools to entertainment. They have been sorted by categoryI hope you enjoy!

Art, Desgin and more

photogradient.com - This website allows you to create your own cool gradients out of photos to use as backgrounds for your desktop or phone. You can add things such as noise. This can only be used for images.

cssgradient.io - This website allows you to create gradients to be used on a website using CSS. I actually used this website to generate the image you see in the background! This can only be used in HTML websites.

coolors.com - This website lets you create colors palettes to use in your next website, project or presentation! It gives the hex codes of the colors.
